## __🎮 Игровая система__
### __💰 Экономика__
В боте реализована внутриигровая экономика, основанная на заработке и трате виртуальной валюты (монет 💰)
### __Основные механики:__
__Заработок монет:__

__Игры:__ Побеждайте в мини-играх (Арбуз, Солнце, Лимонад, Клад, Крестики-нолики, Камень-Ножницы-Бумага) для получения монет и опыта (XP).

__Ежедневные задания:__ Выполняйте два ежедневных квеста, которые обновляются каждые сутки, чтобы получить дополнительные награды.

__Достижения:__ Получайте бонусы при достижении новых уровней (заработок опыта повышает уровень).

#### __Магазин:__

__Титулы:__ Покупка уникальных статусов для профиля (например, 🔥 Профи, 👑 Легенда).

__Кейсы:__ Система лутбоксов (Обычный и Редкий), содержащих случайные награды: монеты, опыт, эксклюзивные эмодзи или титулы.

#### __Аукцион:__

Пользователи могут выставлять заработанные предметы (например, редкие эмодзи из инвентаря) на аукцион и назначать свою цену.

Другие игроки могут покупать эти лоты за монеты.

#### __Инвентарь:__

Все купленные или выигранные редкие предметы (эмодзи) сохраняются в личном инвентаре пользователя, откуда их можно надеть или выставить на продажу.

### __👤 Система аккаунтов__
Бот использует автоматическую систему регистрации, привязанную к уникальному ID вашего профиля в Telegram. Вам не нужно создавать пароли или проходить сложную верификацию.

### __Основные возможности:__

__Автоматическая регистрация__: Ваш аккаунт создается при первом взаимодействии с ботом.

__Привязка к ID:__ Весь прогресс, накопленные монеты и инвентарь сохраняются в базе данных и привязаны строго к вашему Telegram ID.

__Безопасность данных:__ Система использует изолированную базу данных SQLite, что гарантирует сохранность ваших игровых достижений и предотвращает несанкционированный доступ.

__Профиль пользователя:__ Команда /profile позволяет в любой момент просмотреть ваш текущий уровень, количество опыта, баланс монет и список установленных титулов.

__Мультиплатформенность:__ Благодаря тому, что система аккаунтов работает на стороне сервера, ваш прогресс доступен с любого устройства, где установлен Telegram.

### __Глобальная система аккаунтов 🌐__
#### __Ключевые особенности:__

__Единый ID__: Ваш игровой аккаунт, накопленная валюта, титулы и инвентарь привязаны не к конкретному боту, а к вашему глобальному идентификатору, посмотреть ваш глоабльный аккаунт можно через команду /id_profile.

__Кросс-серверный прогресс:__ Вы можете переходить между ботами нашей сети, сохраняя все свои достижения, достижения уровней и редкие предметы.

__Масштабируемость:__ Система спроектирована так, что данные хранятся централизованно, что обеспечивает моментальный доступ к профилю из любого подключенного сервиса.

__Безопасная синхронизация:__ Мы используем уникальные ключи привязки для каждого пользователя, что исключает возможность потери данных при обновлении или расширении сети наших ботов.


### __⚖️ Система прав и статусов__

| Статус | Эмодзи | Права | Описание |
|--------|--------|-------|----------|
| Игрок | 👤 | Базовые | Обычный пользователь |
| Админ | ⭐ | Расширенные | Может банить, выдавать валюту |
| Кодер | 🌐 | Расширенные | Может полностью работать с ботом |
| Разработчик | 🌐💠 | Полные | Владелец бота |
| Забанен | 🚫 | Нет | Доступ заблокирован |

### __📋 Список команд__

```bash
# Доступны для всех
/start - регистрация или/быстрый доспуск к кнопкам которые видут в настройки и тд.
/profile - открывает информацию профиля
/id_profile - открывает информакцию глобальный аккаунта
/sell_emoji - продать эмоджи на аукцион
/help - показывает все команды достпуны для пользователей
/feedback - оставьте приятный коментарий
/support - эта команда в разработке ⚙

# Доступны только для Админов и Разработчиков

/ban @username [Причина] - блокировка пользователя
/unban @useranme - разблокировка пользователя
/id_ban @username - блокировка глобального аккаунта пользователя
/id_unban @username - разблокировка глобального аккаунта пользователя 
/id_freeze - заморозить глобальный аккаунт пользоваетеля
/id_unfreeze - разморозить глобальный аккаунт пользователя


# Управление экономикой 
/give_coins @username [Количество] - изминить баланс пользователя
/take_coins @username [количество] - отнять количество монет у пользователя

# Выдача прав 
(Команды по выдаче парав будут добавленны в следующих патчах)
```
